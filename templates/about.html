About Fitcha

Fitcha is a project to provide a service to predict the size for ladies clothing. The project is a part of the class 'Intelligent Service Development with Big Data' delivered by Samsung Multicampus in Soeul, Korea - an affiliate of Samsung for education - and operated by Korean Ministry of Employment and Labor as a K-digital training scheme. This project mainly focuses on building a service from the scratch utilizing machine learning, database and web application skills.

The aim of this project is to help customers find their size for online shopping to minimize the return rate and maximize the satisfaction of purchases, especially for Korean people shopping often overseas. It recommends clothing size predicted by machine learning model which was built up on the data sourced from Kaggle. The data include age, height and weight information, and the model explored which features work best for prediction. If you would like to check the data, the link can be found at the footer 'Quick Links'.

Not only just provide size recommendation, it suggests shopping webpages which will direct a customer to the clothes of the recommended size so that it serves its original purpose to improve users' shopping experience.

The downside of the data is that the source is not explicitly mentioned. Plus, it is also not mentioned if it is a preferred size or an exact size of each person which makes a big difference in outcome. To improve the service, customers can provide their actual size if the prediction is different from it. Then the model can learn from the right size, and the shopping pages directed from Fitcha show users clothes with the right size, rather than the recommended size.
